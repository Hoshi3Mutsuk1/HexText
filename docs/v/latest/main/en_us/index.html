      Parse Purification (str | [str] → text)

      • Renders
        • Palettes
          Your browser does not support visualizing patterns. Pattern code: awdwadeewwee

          Remove a string or string list from the stack and returns the Text that string parses as (or Empty Text if it doesn't parse).


          Moniker Purification (entity → text)

          • Renders
            • Palettes
              Your browser does not support visualizing patterns. Pattern code: qaqwewdew

              Takes an entity and returns that entity's name. If it is an item entity, return the item's name.


              Name (entity, text →)

              • Renders
                • Palettes
                  Your browser does not support visualizing patterns. Pattern code: edewwqawq

                  Takes a Text and an entity, and sets that entities name to the given Text. If the entity is an item entity, rename the item.
